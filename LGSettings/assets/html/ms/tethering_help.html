Penambat USB

Anda boleh mengikat peranti Android anda bagi komputer anda dengan kabel USB, berkongsi hubungan Internet peranti anda dengan komputer anda
  • Penambat USB berfungsi dengan Windows Vista, Windows 7 , dan Linux

  • Jika peranti anda mempunyai kad SD atau USB storan, anda tidak boleh mengikatkan pada komputer anda apabila USB ditambat

  • Untuk maklumat lanjut, termasuk menggunakan penambat USB dengan OSs lain, lihat http://www.android.com/tether#usb

Hotspot mudah alih Wi-Fi

Anda boleh menukar peranti Android anda ke satu hotspot mudah alih Wi-Fi, berkongsi hubungan Internet peranti Android anda dengan satu atau lebih komputer-komputer atau alat-alat lain
  • Bila peranti anda berkhidmat sebagai hotspot Wi-Fi, anda tidak boleh menggunakan aplikasi telefon anda untuk laluan Internet melalui sambungan Wi-Finya (tetapi sudah tentu anda kekal berhubung kepada Internet melalui data bergerak anda)

  • Anda mengkonfigurasi hotspot dengan tetapan hotspot Wi-Fi

  • Untuk maklumat lanjut, lawati http://www.android.com/tether#wifi
