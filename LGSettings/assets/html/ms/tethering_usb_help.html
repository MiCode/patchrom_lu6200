Penambat USB

Anda boleh mengikat peranti Android anda bagi komputer anda dengan kabel USB, berkongsi hubungan Internet peranti anda dengan komputer anda
  • Penambat USB berfungsi dengan Windows Vista, Windows 7 , dan Linux

  • Jika peranti anda mempunyai kad SD atau USB storan, anda tidak boleh mengikatkan pada komputer anda apabila USB ditambat

  • Untuk maklumat lanjut, termasuk menggunakan penambat USB dengan OSs lain, lihat http://www.android.com/tether#usb
