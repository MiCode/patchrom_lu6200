USB tethering

Anda dapat menambatkan perangkat Android Anda ke komputer menggunakan kabel USB, untuk berbagi koneksi internet perangkat dengan komputer Anda
  • USB tethering dapat berfungsi pada Windows Vista, Windows 7, dan Linux

  • Jika perangkat anda terdapat kartu SD pada penyimpan USB, Anda tidak dapat memasangnya pada komputer ketika USB ditambatkan

  • Untuk informasi yang lebih jelas, termasuk menggunakan USB tethering pada sistem operasi lain, silahkan kunjungi http://www.android.com/tether#usb
