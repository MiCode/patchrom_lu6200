USB 數據連線

您可以使用 USB 連接線讓手機和電腦連線，使電腦可以共用手機的網絡連線
  • 支援數據連線的作業系統包括 Windows Vista、Windows 7 及 Linux

  • 使用 USB 數據連線時，您無法將手機的 SD 卡連接到電腦上

  • 更多資訊（包括使用 USB 數據連線到其他操作系統），請參見 http://www.android.com/tether#usb

可攜式 Wi-Fi 熱點

您可以將手機變成 Wi-Fi 存取點，讓一或多台電腦或其他裝置可以共用手機的網絡連線
  • 將手機當作存取點使用時，手機的應用程式將無法透過 Wi-Fi 連線存取網絡

  • 您可以在 Wi-Fi 數據連線設定中配置 Wi-Fi 存取點

  • 更多資訊，請連結至 http://www.android.com/tether#wifi
