USB tethering

您可以使用 USB 數據連接線讓手機和電腦連接，使電腦可以共用手機的網際網絡連線
  • 支援數據連線的作業系統包括 Windows Vista, Windows 7 及 Linux

  • 當 USB 數據連線時，您無法將手機 SD 卡插到電腦上

  • 如需更多資訊，包括在其他作業系統中使用 USB 數據連線的資訊，請參閱 http://www.android.com/tether#usb

可攜式 Wi - Fi 無線熱點

您可以將手機變成可攜式 Wi-Fi 無線熱點，讓一台或多台電腦或其他裝置可以共用手機的網際網絡連線
  • W將手機當作 Wi-Fi 無線熱點時，手機應用程式將無法透過 Wi-Fi 連線存取網際網絡

  • 您可在 Wi-Fi 無線熱點設定中配置無線熱點

  • 如需更多資訊，請瀏覽 http://www.android.com/tether#wifi
