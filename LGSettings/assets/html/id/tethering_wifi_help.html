Hotspot Wi-Fi portable

Anda dapat mengubah perangkat Android anda menjadi hotspot Wi-Fi portable, untuk berbagi koneksi internet perangkat dengan satu komputer atau lebih atau perangkat lainnya
  • Ketika perangkat Anda sedang berfungsi sebagai hotspot Wi-Fi, anda tidak dapat menggunakan aplikasi telepon untuk mengakses Internet melalui koneksi Wi-Fi nya (Tetapi tentu saja Anda tetap terhubung ke Internet)

  • Anda mengkonfigurasikan hotspot melalui pengaturan hotspot Wi-Fi

  • Untuk informasi lebih lanjut, silahkan kunjungi http://www.android.com/tether#wifi
