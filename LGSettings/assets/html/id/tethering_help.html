USB tethering

Anda dapat menambatkan perangkat Android Anda ke komputer menggunakan kabel USB, untuk berbagi koneksi internet perangkat dengan komputer Anda
  • USB tethering dapat berfungsi pada Windows Vista, Windows 7, dan Linux

  • Jika perangkat anda terdapat kartu SD pada penyimpan USB, Anda tidak dapat memasangnya pada komputer ketika USB ditambatkan

  • Untuk informasi yang lebih jelas, termasuk menggunakan USB tethering pada sistem operasi lain, silahkan kunjungi http://www.android.com/tether#usb

Bluetooth tethering

Anda dapat menambatkan perangkat Android Anda melalui koneksi Bluetooth, untuk berbagi koneksi internet perangkat dengan komputer Anda
  • Bluetooth tethering dapat berfungsi pada tiap komputer atau perangkat lain yang mampu mendapatkan koneksi Internet melalui Blooetooth

  • Untuk Informasi lebih lanjut, silahkan kunjungi http://www.android.com/tether#bluetooth

Hotspot Wi-Fi portable

Anda dapat mengubah perangkat Android anda menjadi hotspot Wi-Fi portable, untuk berbagi koneksi internet perangkat dengan satu komputer atau lebih atau perangkat lainnya
  • Ketika perangkat Anda sedang berfungsi sebagai hotspot Wi-Fi, anda tidak dapat menggunakan aplikasi telepon untuk mengakses Internet melalui koneksi Wi-Fi nya (Tetapi tentu saja Anda tetap terhubung ke Internet)

  • Anda mengkonfigurasikan hotspot melalui pengaturan hotspot Wi-Fi

  • Untuk informasi lebih lanjut, silahkan kunjungi http://www.android.com/tether#wifi
